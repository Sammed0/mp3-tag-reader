# mp3-tag-reader

MP3 Tag Reader in C  This project is a simple MP3 Tag Reader developed using the C programming language. It reads and displays metadata (ID3 tags) from MP3 files such as title, artist, album, and other details.

Features:
1.Reads MP3 file metadata
2.Displays song information (Title, Artist, Album)
3.Uses file handling to extract tag data

Concepts Used:
1.File handling in C
2.Structures
3.Pointers
4.Binary file processing

How to Run:
1.Compile the program:
gcc main.c
2.Run the executable:
./a.out <mp3_file>

Output:
Displays metadata information of the given MP3 file.
